Analyse des donnes

On introduit une fonction quon va rutiliser tout au long de notre analyse. Elle permet de rcuprer la proportion de clients faisant dfaut pour chaque catgorie dune variable qualitative.

TableProp = function(data, y, x){
  n <-  dim(data)[1]
  n_x <-  table(data[x])
  names_x <-  names(table(data[x]))
  df <-  data.frame(numeric(2))
    
  for(j in 1:length(n_x)){
    df[names_x[j]] <- table(data[data[x] == names_x[j], y])/n_x[j]
  }
  df = df[-1]
  rownames(df) <- names(table(data[y]))
    
return(df)
}

Le F1-Score est calcul partir de la prcision et du rappel (recall). Il sagit enfait dune moyenne harmonique de ces deux termes.

Autrement dit, dans notre cas, la prcision mesure la proportion de clients correctement identifie comme ayant fait dfaut par le modle sur le nombre total de clients prdits comme ayant fait dfaut.

On se focalise sur le rappel car, si on se place du point de vue dune banque, on veut que le taux de faux ngatif soit le plus bas possible. En dautres termes, on veut tre capable didentifier correctement le plus de clients susceptibles de faire dfaut.
